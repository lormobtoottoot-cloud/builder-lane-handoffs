## Safety Rules

This repository is static and mock-only.

Do not add:

- secrets, passwords, private tokens, or API keys
- live trading
- wallet movement
- wallet, vault, bank, or security modules
- backend trading logic
- terminal execution
- CORE Vault files

Allowed work:

- mock UI
- starter pages
- instructions
- CSS polish
- disabled buttons
- fake data
- notes and checklists
