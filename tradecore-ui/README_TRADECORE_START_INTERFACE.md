Provide a clean TradeCore interface mock that is completely separate from CORE Vault and contains no real trading logic.

- Header: TradeCore
- Risk warning banner
- Mock portfolio summary

- Mock trading chart area
- Mock Coinbase connection card
- Multi-platform connector preview
- AI agent panel
- Recent mock transactions
- Settings/API safety screen
- Mobile-first layout

## Safety

- No backend.
- No server.
- No real API calls.
- No Coinbase keys.
- No wallet movement.
- No real trades.
- Buy/sell buttons are disabled and labeled mock-only.
