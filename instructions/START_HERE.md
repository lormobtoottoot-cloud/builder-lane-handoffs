# Start Here

This repo is a public builder portal for two safe starter lanes:

- Spark Builder Lane for Aunt
- TradeCore UI Builder Lane for younger aunt / sister

Open the portal homepage first:

https://lormobtoottoot-cloud.github.io/builder-lane-handoffs/

## Main Paths

1. Click a live interface and explore.
2. Download a zip and build/edit locally.
3. Read the instructions for your lane before touching files.

## Safety

Everything here is static and mock-only. Do not add secrets, API keys, backend logic, terminal logic, wallet logic, or trading execution.
