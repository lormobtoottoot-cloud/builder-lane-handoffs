Section 3

How To Use

For Spark builder

  1. Click Open Spark Interface.
  2. Read the screen.
  3. Do not rewrite the app.
  4. Do not convert to React/Vite.
  5. Use the mock buttons only.
  6. Wait for next assignment before editing production Spark files.

For TradeCore UI builder

  1. Click Open TradeCore Interface.
  2. Explore the mock dashboard.
  3. Improve layout, spacing, text, and mobile design only.
  4. Use mock data only.
  5. Do not connect Coinbase.
  6. Do not add real buy/sell behavior.
